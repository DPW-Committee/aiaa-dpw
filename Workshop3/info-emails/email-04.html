There are some new grids available on the ftp download site (https://dpw.larc.nasa.gov/DPW3). There are additional structured grids in the Boeing multiblock family and additional unstructured node-based grids from NASA Langley.

Please be sure to view the README file in the DPW3 directory for a history of the grid updates .

We do apologize that the grids have not been evaluated as well as they have in the past. Please let us know if you discover any issues with any of the grids.

There are currently unresolved issues with the following grids:

hex_Embraer - dpw3_DLRF6_fairingoff_24000kcells.tar.gz - crossed cells
multiblock_icem_cfd/w2 grids - surface spacing
multiblock_icem_cfd/w1 medium grid - missing blocks

I also apologize that I have not been sending out timely notifications on the grid updates. I will try to do better in the future.
