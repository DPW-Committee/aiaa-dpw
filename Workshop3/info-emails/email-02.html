Abstract extension - Feb 10
Please identify the geometries for which you intend to present solutions (DLR-F6 and/or Wing Alone).

The moment reference information on the DPW Wing Alone geometries was found to be in error. It will be updated on the web site soon, but here it is:

Relative to the wing root leading edge:
x=154.245 mm, z=0.0 mm

The W2 IGES geometry has the wing root leading edge at the origin, so the above is the correct moment reference. For the W1 IGES geometry the moment reference translates to and z=2226.615 mm.

Please note that some grids may move the geometry, so you should verify the wing root leading edge location (e.g. the Boeing DPW-W1 grid has been translated so the wing root leading edge is at the origin).

Errors have been found in some of the grids. Please check the ftp site for updates.
