Data Submittal Forms and Postprocessing Information

Overview

DPW-8/AePW-4 is collecting data in a Tecplot-readable ASCII format using the templates below. Please complete as much of each form as possible, including the auxiliary data at the top of each form identifying the dataset. Please complete as much of each form as possible, including the auxiliary data at the top of each form identifying the dataset. Forms are provided for a variety of data sets. Coordinate with your Working Group leaders regarding which files must be submitted for that group's test cases.

  • Vehicle integrated forces and moments
  • Surface pressure (macro provided below)
  • Standard views and contour plots (macro provided below)
  • Spanwise load distribution (more to come later)
  • Boundary layer profiles (more to come later)

Data will be submitted to the DPW/AePW github site. Process overview, Submission Tutorial, and GitHub Submission Repository

Tecplot has graciously agreed that all participants who do not have access to Tecplot may request a free extended evaluation license of Tecplot 360 for use in the AIAA DPW and AePW. If you are interested, please email dpwaiaa@gmail.com.

Contact dpwaiaa@gmail.com prior to submitting any data. The Organizing Committee will assign you a participant ID. You will then use that in the files that you submit data directly to your working group leaders.

Some data are optional (see inside the data files). Use -999 for any optional data that you are not submitting.
